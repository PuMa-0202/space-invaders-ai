# Space Invaders (AI Generated)

Це класична аркадна гра Space Invaders, створена повністю за допомогою штучного інтелекту.

## 🎮 Як грати
1. Завантажте файл `index.html` або відкрийте його.
2. Відкрийте файл у будь-якому сучасному браузері.

**Управління:**
- ⬅️ / ➡️ або **A / D**: Рух корабля
- **Пробіл**: Постріл
- **Enter**: Перезапуск гри

## 🤖 Процес створення
Використаний AI: Gemini Pro

**Як це було зроблено:**
1. Я надав агенту технічне завдання з файлу AGENTS.md.
2. Отримав базову версію гри.
3. Попросив виправити баг зі звуковим супроводженням.
4. Додав додаткові фічі: додано ефекти тряски екрану, коли корабель вибухає. Додано різні кольори на кожен ряд прибульців. Швидкість гри поступово зростає з кожним рівнем. Додано прості звукові ефекти, використовуючи Web Audio API, щоб не вантажити зовнішні файли.
